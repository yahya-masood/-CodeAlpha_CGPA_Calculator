#  CGPA Calculator (C++)

##  Overview
This project is a simple yet efficient CGPA Calculator developed in C++. It allows users to input course details, calculate GPA for a semester, and compute the overall CGPA.

##  Features
- Input number of courses
- Enter grades and credit hours for each course
- Calculates:
  - Total Credit Hours
  - Total Grade Points
  - Semester GPA
  - Overall CGPA
- Displays complete result clearly

##  Technologies Used
- C++

##  Concepts Applied
- Loops and Conditional Statements
- Basic Mathematical Computation
